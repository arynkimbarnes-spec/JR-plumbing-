# JR Plumbing & Electrical — Website

Simple static website for JR Plumbing & Electrical (Cape Town).

## What this includes
- `index.html`, `about.html`, `services.html`, `contact.html` — simple site pages
- `css/styles.css` — styling with local fonts
- `js/script.js` — small frontend scripts
- `images/` — place the two images you uploaded here (`toilet.jpg`, `leak.jpg`)
- `fonts/` — place your font files here (woff/ttf/otf)
